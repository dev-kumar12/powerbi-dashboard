# Power BI Sales Analytics Dashboard

## Overview
Interactive Power BI dashboard demonstrating sales performance analysis built with Power BI Desktop and an Excel dataset. Includes KPIs, trend analysis, region/product breakdowns, and drill-through reports to support data-driven decisions.

## Files
- `your_dashboard.pbix` — Power BI Desktop report file (open in Power BI Desktop).
- `dataset.xlsx` — Source data (Excel) used by the report.
- `README.md` — Project overview and instructions.

## How to open
1. Download `your_dashboard.pbix` and `dataset.xlsx` to the same folder.
2. Open `your_dashboard.pbix` with Power BI Desktop (latest version recommended).
3. If prompted to locate the dataset, point the data source to `dataset.xlsx` via **Transform data → Data source settings**.
